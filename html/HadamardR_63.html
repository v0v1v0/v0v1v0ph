pow

Description

pow functions finds the exponent of 2.

Usage

pow(n)

Arguments

n

integer

Details

This function checks the given number is the power of 2 or not If the given number is power of 2 it returns the exponent value; otherwise NULL is returned.

Value

power of 2

Examples

pow(4)
# 2
pow(5)
#NULL
pow(6)
#NULL
