tmr_title<- R Documentation

Set Title hms Timer

Description

Sets the title of a hms_timer().

Usage

tmr_title(x) <- value

Arguments

x

A hms_timer().
value

A string of the title.

Value

A copy of the hms_timer() with the new title.

See Also

tmr_title()

Examples

tmr <- tmr_timer(title = "A title")
tmr_print(tmr)
tmr_title(tmr) <- "A different title"
tmr_print(tmr)
tmr_title(tmr) <- NULL
tmr_print(tmr)
