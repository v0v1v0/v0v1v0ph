toy R Documentation

Toy Dataset

Description

A made-up (toy) dataset.

Usage

data(toy)

Format

This data frame consists of 2 made-up variables for a sample of size 5:

  • x The made-up x values.

  • y The made-up y values.

Source

Young, D. S. (2017), Handbook of Regression Methods, CRC Press.
