printsplit R Documentation

Displays split table for Fitted htt Object

Description

Display the split table for fitted htt object.

Usage

printsplit(object)

Arguments

object

fitted model object of class htt returned by the HTT function.

Value

Display the split table.

See Also

HTT, htt.object

Examples

irishtt = HTT(Species ~., data = iris)
printsplit(irishtt)
