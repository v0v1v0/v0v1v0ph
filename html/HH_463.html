pdiscunif R Documentation

Discrete Uniform Distribution

Description

Discrete Uniform Distribution

Usage

pdiscunif(q, size)
qdiscunif(p, size)
ddiscunif(q, size)
rdiscunif(n, size)

Arguments

size

parameter of distribution. Numbers from 1 to size are equally likely.
q

Quantiles.
p

Probability.
n

number of items in the random sample.

Author(s)

Richard M. Heiberger <rmh@temple.edu>

Examples

  q <- seq(-.5, 7.5, .5)

  pp <- pdiscunif(q, 6)

  ## xyplot(pp ~ q,
  ##        scales=list(
  ##        x=list(at=floor(min(q)):ceiling(max(q))),
  ##        y=list(at=seq(0, 1, .1))))

  qq <- qdiscunif(pp, 6)

  dd <- ddiscunif(q, 6)

  cbind(q, pp, qq, dd)

  rdiscunif(12, 6)
