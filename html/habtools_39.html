scale_volume R Documentation

Re-scale mesh based on a fixed volume of 1

Description

Re-scale mesh based on a fixed volume of 1

Usage

scale_volume(mesh)

Arguments

mesh

A triangular mesh of class mesh3d.

Value

A mesh with volume = 1.

Examples

Rvcg::vcgVolume(mcap)
mcap_scaled <- scale_volume(mcap)
Rvcg::vcgVolume(mcap_scaled)
