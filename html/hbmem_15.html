sd2 R Documentation

Description

Signal detection dataset that hasn't been published yet. Please do not use.

Usage

data(sd2)

Format

A data frame with observations on the following 6 variables.

cond

a numeric vector

Scond

a numeric vector

sub

a numeric vector

item

a numeric vector

lag

a numeric vector

resp

a numeric vector
