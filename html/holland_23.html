example3 R Documentation

RIASEC and Big-Five correlations - perfect

Description

another fictional correlation matrix representing RIASEC (circumplex) correlations and correlations (with) five dimensions of personality (big-five).

Usage

  data(example3)

Format

A 11 x 11 matrix object with named columns and rows.

Details

can be written as a .dat text file using the function write_dat().

References

Holland, J.L. (1997). Making vocational choices. A theory of vocational personalities and work environments. Lutz, FL: Psychological Assessment Resources.

Examples

data(example3)
