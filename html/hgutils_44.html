messages R Documentation

Display coloured messages

Description

Display coloured messages

Usage

mess_info(header, message, pre = "")

mess_succ(header, message, pre = "")

mess_fail(header, message, pre = "")

mess_warn(header, message, pre = "")

mess_numb(header, message, pre = "")

mess_hint(message, header = "", pre = "")

Arguments

header

message title.
message

the message.
pre

optional string to be displayed before the message bullet (such as '\n').

Value

A coloured message.
