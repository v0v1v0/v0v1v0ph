uptake_plots R Documentation

Uptake plots

Description

Uptake plots per peptide

Usage

uptake_plots(
  input_data,
  timepoints,
  replicates = 3,
  cola = NA,
  seq_match = TRUE
)

Arguments

input_data

output from function output_tp(..., percent=T)
timepoints

the labeling times
replicates

replicates
cola

colors, default NA
seq_match

Flag TRUE or FALSE, default TRUE, match sequence of the protein states

Value

Uptake plots

Examples


file_nm<-system.file("extdata", "All_results_table.csv", package = "HDXBoxeR")
a<- output_tc(file_nm, percent=TRUE)
x=c(3,60, 1800, 72000)
uptake_plots(a, x)
