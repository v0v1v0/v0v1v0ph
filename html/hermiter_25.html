hermite_int_full R Documentation

Convenience function to output the integral of the orthonormal Hermite functions on the full domain

Description

The method calculates \int_{-\infty}^{\infty} h_k(t) dt for k=0,\dots,N.

Usage

hermite_int_full(N)

Arguments

N

An integer number.

Value

A numeric matrix with N+1 rows and 1 columns.
