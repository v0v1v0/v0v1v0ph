chk.hy R Documentation

Validation of hyperSpec objects

Description

Check whether an object is a hyperSpec object and validate the object

Usage

chk.hy(object)

Arguments

object

the object to check

Value

TRUE if the check passes, otherwise stop with an error.

Author(s)

C. Beleites

See Also

Examples

chk.hy (chondro)
validObject (chondro)
