sf.tiles R Documentation

Obtain projected versions of sf.tiles_elbe and sf.tiles_rhine

Description

Obtain projected versions of sf.tiles_elbe and sf.tiles_rhine

Usage

sf.tiles(name = NULL)

Arguments

name

either 'Elbe' or 'Rhine'.

Value

sf with projected tiles

See Also

sf.tiles_elbe, sf.tiles_rhine

Examples

  library(hydflood)
  sf.tiles(name = "Elbe")
