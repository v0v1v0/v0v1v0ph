Jmat R Documentation

Jmat

Description

Jmat peforms the generation of unit matrix.

Usage

Jmat(n)

Arguments

n

integer

Details

An J matrix is a square matrix where all the entries are one.

Value

square matrix of order n

Examples

Jmat(4)
#      [,1] [,2] [,3] [,4]
#[1,]    1    1    1    1
#[2,]    1    1    1    1
#[3,]    1    1    1    1
#[4,]    1    1    1    1
