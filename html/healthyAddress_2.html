.digit256 R Documentation

Extract the n-th digit of a duocentehexaquinquagesimal number

Description

Extract the n-th digit of a duocentehexaquinquagesimal number

Usage

.digit256(x, d)

Arguments

x

integer(n)
d

integer(1) One of 0:3. Other integers silently return x.

Value

For b = 256 if

x = a_0 + a_1b + a_2b^2 + a_3b^3

then .digit(x, d) = a_d
