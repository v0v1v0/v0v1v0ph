SE R Documentation

Extract the standard errors from a var-cov matrix.

Description

Extract the standard errors from a var-cov matrix.

Usage

SE(object, ...)

Arguments

object

est.beta object
...

unused

Examples

df <- make.randomized.dat( 1000, gamma.vec=c(1,1,1,2), beta.vec=c(-1,-1,1,0) )
es <- estimate_systematic( Yobs ~ Z,  interaction.formula = ~ A + B, data = df )
SE(es)
