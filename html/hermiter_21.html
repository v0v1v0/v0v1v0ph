Description

Convenience function to output the sum of orthonormal Hermite functions

The method calculates the sum of orthonormal Hermite functions, \sum_{i} h_k(x_{i}) from k=0,\dots,N for the vector of values, x.

Usage

hermite_function_sum_N(N, x)

Arguments

N

An integer number.
x

A numeric vector.

Value

A numeric vector of length N+1.
