Description

Obtain projected versions of sf.afe and sf.afr

Usage

sf.af(name = NULL)

Arguments

name

either 'Elbe' or 'Rhine'.

Value

sf with the projected active floodplain

See Also

sf.afe, sf.afr

Examples

  sf.af(name = "Elbe")
