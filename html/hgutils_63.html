rm_empty_rows R Documentation

Remove empty rows

Description

Remove empty rows

Usage

rm_empty_rows(dataframe)

Arguments

dataframe

data.frame object.

Value

A data.frame with rows removed that only contain NA.

See Also

Other NA functions: rm_na()

Examples

data <- rbind(c(1,2,3), c(1, NA, 4), c(4,6,7), c(NA, NA, NA), c(4, 8, NA))
rm_empty_rows(data)
