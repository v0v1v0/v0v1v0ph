Description

Returns the key(s) from a hash

Usage

  keys(x)

  ## S3 method for class 'hash'
names(x)

Arguments

x

A hash object.

Details

Returns the character vector containing the keys of a hash object.

Value

keys

A vector of type character

Author(s)

Christopher Brown

See Also

See Also hash.

Examples


  h <- hash( letters, 1:26 )
  keys(h)  # letters

  names(h) # same
