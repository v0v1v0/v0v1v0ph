AIST_2005_F_1270 R Documentation

RIASEC correlations - female AIST norm sample

Description

a matrix of correlation coefficients based on n = 1270 observations, obtaind from the german female norm sample of the AIST published in Bergman & Eder (2005).

Usage

  data(AIST_2005_F_1270)

Format

A 6 x 6 matrix object with named columns and rows.

Details

can be written as a .dat text file using the function write_dat().

References

Bergmann, C.,& Eder, F. (2005). AIST-R Allgemeiner Interessen-Struktur-Test - Revision. Goettingen: Hogrefe.

Examples

data(AIST_2005_F_1270)
