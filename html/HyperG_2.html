hypergraph.add.edges R Documentation

Add edges or vertices to a hypergraph.

Description

Adds hyper-edges or vertices to a hypergraph.

Usage

hypergraph.add.edges(h, edges, verbose = FALSE)
add.hyperedges(h, edges, verbose = FALSE)
hypergraph.add.vertices(h, nv, names)

Arguments

h

A hypergraph.

edges

A list of edges to be added.

nv

Number of vertices to add.

names

Optional vector of names of the vertices.

verbose

logical. Whether to warn if new vertices are created.

Details

The edges can be indices or edge names. This is different than the graph call â€“ see the igraph package help for that. If edges is NULL, or missing, hypergraph.add.edges adds a single empty hyper-edge to the hypergraph. add.hyperedges is an alias for hypergraph.add.edges.

Value

Returns a hypergraph (or graph) as appropriate.
