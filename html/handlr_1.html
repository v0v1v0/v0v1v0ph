handlr-package R Documentation

Citation format converter

Description

A tool for converting among citation formats

supported readers

  • citeproc

  • ris

  • bibtex (requires suggested package bibtex)

  • codemeta

  • cff

supported writers

  • citeproc

  • ris

  • bibtex

  • schema.org

  • rdfxml (requires suggested package jsonld)

  • codemeta

  • cff
