hql_file R Documentation

Access HDF Files

Description

Open (use) and close HDF files.

Usage

hql_use_file(file)

hql_close_file(file, all = FALSE)

Arguments

file

The HDF file path.
all

If TRUE, close all open HDF files.

Functions

  • hql_use_file: Open (use) an HDF file.

  • hql_close_file: Close an HDF file.

Examples

if(hql_is_loaded()){
  tf = tempfile(fileext = ".h5")
  hql_create_file(tf)

  hql_use_file(tf)
  hql_flush()

  hql_close_file(tf)
}
