tmr_ceiling R Documentation

Ceiling hms Timer

Description

Rounds a hms_timer() up to the nearest second.

Usage

tmr_ceiling(x)

Arguments

x

A hms_timer().

Value

A hms_timer().

See Also

Other round: tmr_floor(), tmr_format(), tmr_round()

Examples

tmr_ceiling(tmr_timer(18.9))
tmr_ceiling(tmr_timer(122.1))
