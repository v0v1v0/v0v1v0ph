Return Log Likelihood of a fit generated by LD50

Description

Return Log Likelihood of a fit generated by LD50

Usage

Arguments

object

A result file generated by fitRMU
...

Not used

Details

logLik.LD50 Return Log Likelihood of a fit for LD50

Value

The Log Likelihood value for the fitted model with data
