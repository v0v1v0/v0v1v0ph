mmstat.math R Documentation

mmstat.math

Description

Returns a math expression based on HTML special characters notation.

Usage

mmstat.math(txt)

Arguments

txt

character: input text

Value

expression

Examples

mmstat.math(" &bar(X);~&N(mu[0], sigma^2/n); ")
mmstat.math("&H[0];: &mu==mu[0]; vs. &H[1];: &mu!=mu[0]; ")
