Mt0-class R Documentation

Class "Mt0"

Description

"Mt0" contains the hyperprior values of the seasonal component and it is an internal class stored in "Parameters0".

Slots

seas0:

A "list" of objects of class "Seas0", one for each seasonality of the model.

Methods

[

signature(x = "Mt0", i = "character", j = "missing", drop = "missing"): extract the components of the model.

[<-

signature(x = "Mt0", i = "character", j = "missing"): assign values to the components of the model.
