as.num R Documentation

Converts vector of strings to numeric vector

Description

Converts vector of strings to numeric vector

Usage

as.num(x, na.strings = "NA")

Arguments

x

Input vector of strings.
na.strings

A string which represents NA. Default: "NA"

Value

A numeric vector

Examples

library(haploR)
as.num(c("1", "2", "X"), na.strings="X")
